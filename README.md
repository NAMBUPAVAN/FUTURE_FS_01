# 🌟 Premium Portfolio Website

A modern, responsive, and interactive personal portfolio website built using **HTML5, CSS3, and JavaScript**.

## 📌 Features

- 🎨 Modern Premium UI Design
- 📱 Fully Responsive Layout
- 🌙 Dark Theme
- 🚀 Smooth Scrolling
- 📌 Sticky Navigation Bar
- 📂 Projects Showcase
- 👨‍💻 About Section
- 🛠 Skills Section
- 📞 Contact Form
- 🔝 Back-to-Top Button
- ✨ Scroll Reveal Animation
- 🎭 Animated Typing Text
- 📅 Automatic Copyright Year
